## Getting Started

### Instalação

1. Clonar o repositório
   ```sh
   git clone https://github.com/WeesleyRosa/assembly.git
   ```
2. Rodar o docker compose
   ```sh
   docker-compose up -d
   ```
3. Iniciar a aplicação normalmente
